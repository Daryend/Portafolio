
## Portafolio Web

## Descripción General
Este es un sitio web de portafolio creado por Daryend Castillo, un desarrollador en formación con sede en Riobamba, Ecuador. El sitio muestra diversos proyectos realizados durante mi etapa académica en la ESPOCH. Mi objetivo es convertirme en un gran desarrollador, y proyectos como este son pasos clave en mi camino de aprendizaje y crecimiento profesional.

# Archivos
- index.html: El documento HTML principal. Incluye secciones de bienvenida, una lista de proyectos y un pie de página con información de contacto.
- styles.css: Contiene los estilos visuales del portafolio, implementando un diseño futurista y estilo cyberpunk con una paleta de colores vibrantes sobre una base contrastada.
- script.js: Archivo destinado a la funcionalidad interactiva, animaciones y cualquier lógica dinámica del sitio.

# Características
- Diseño limpio, moderno y con estética vanguardista.
- Diseño responsivo (adaptable a diferentes dispositivos y pantallas).
- Galería de proyectos con descripciones técnicas.
- Información de contacto integrada en el pie de página.

# Instrucciones de Configuración
1. Clona el repositorio en tu máquina local.
2. Abre el archivo index.html en tu navegador para visualizar el portafolio.
3. Modifica el archivo styles.css para personalizar el diseño según tus preferencias.
4. Añade cualquier funcionalidad adicional en el archivo script.js.
